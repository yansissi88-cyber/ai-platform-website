## 🎨 视觉风格指南

### 配色方案
- **主背景**: #020204 (深黑)
- **卡片背景**: #12121a (暗灰)
- **主色调**: #3b82f6 (蓝) → #8b5cf6 (紫) → #ec4899 (粉)
- **强调色**: #10b981 (成功绿)、#f59e0b (警告橙)
- **文字**: #ffffff (主文字)、#94a3b8 (次文字)

### 字体
- **标题**: Inter Bold
- **口播字幕**: Inter Medium + 描边
- **数据**: JetBrains Mono

### 动效原则
- 所有动画使用 ease-out 缓动
- 转场时长：0.3-0.5秒
- 呼吸动画周期：2-3秒
- 数据滚动：1-2秒

## 📹 录制指南

### 录制工具推荐
1. **Screen Studio** (Mac) - 自动缩放、平滑过渡
2. **OBS Studio** (Win/Mac/Linux) - 免费专业
3. **Camtasia** - 后期编辑强大
4. **Loom** - 快速录制

### 录制步骤
1. 打开 Kira 网站（或本地演示环境）
2. 设置浏览器窗口为 1920x1080
3. 按分镜逐个录制片段
4. 使用 OBS 场景切换实现无缝转场

### 后期制作
1. **剪辑**: 确保每个镜头时长精确
2. **调色**: 统一深色科技风格
3. **字幕**: 添加动态字幕（关键卖点）
4. **音效**: 添加转场音效和背景音乐
5. **导出**: H.264, 1080p, 60fps
